# Contributing

This is a community guide to working well with Claude Code. Improvements are welcome — corrections, clearer explanations, new hard-won tips, translations.

## How to contribute

1. Fork the repository.
2. Make your change on a branch.
3. Open a pull request with a short description of what and why.

## What makes a good addition

- **Specific and actionable.** "Be careful with context" is weak; "run `/clear` when switching to an unrelated task" is useful.
- **Distilled from real use.** Tips that come from actually hitting a problem and solving it.
- **Tool-accurate.** Claude Code evolves. If you describe a command or feature, make sure it reflects current behavior. When unsure, check the official docs: <https://docs.claude.com/en/docs/claude-code>.
- **Free of anything personal or proprietary.** No real credentials, internal hostnames, client names, or private project details — in examples use placeholders.

## Style

- Plain, direct English. Short sentences. Concrete before abstract.
- Keep each guide focused on one topic; link to others rather than repeating.
- Code blocks for anything you'd type; tables for command references.

## Translations

Translations (the README already carries a Russian quick-start) are welcome. Keep them in sync with the English source when the source changes.

Thanks for helping make this better.
